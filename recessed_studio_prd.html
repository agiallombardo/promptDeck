# Recessed Studio ## Product Overview **The Pitch:** PresCollab is a secure, collaborative environment for reviewing AI-generated, cybersecurity-focused presentation slides. It isolates the presentation as the absolute focal point while pushing collaboration tools into a visually recessed, distraction-free dark interface. **For:** Security analysts, account managers, and clients who need to review, annotate, and approve sensitive presentation materials without UI clutter. **Device:** desktop **Design Direction:** A structured, ultra-dark "Recessed Studio" where side panels and navigation are visually sunken using deep background colors and inner shadows, elevating the central slide canvas. **Inspired by:** Figma (Dark Mode), Splunk's dashboard environments. --- ## Screens - **Presentation Canvas:** Central hub displaying the active slide with recessed navigation. - **Feedback Sidebar:** Sunken right-hand panel for real-time threaded annotations. - **Version History Drawer:** Left-hand recessed panel showing timeline of AI generations. - **Share Modal:** Focused overlay for managing viewing permissions and selective audiences. - **Export Configurator:** Dialog for compiling specific slides into a downloadable PDF. --- ## Key Flows **Review and Annotate:** Collaborators leaving feedback on a specific slide generation. 1. User is on Presentation Canvas -> sees the elevated slide center-stage. 2. User clicks `Leave Feedback` button -> Feedback Sidebar slides in from the right. 3. User clicks directly on the slide canvas -> A highly visible marker drops, tying the comment to coordinates. 4. User types in the recessed sidebar input and hits `Enter` -> Comment appears in the real-time thread. **Force Refresh Generation:** Requesting a new AI iteration based on feedback. 1. User is on Presentation Canvas -> sees current slide with resolved comments. 2. User clicks `Force Refresh` in top nav -> Screen darkens, loading skeleton pulses. 3. New iteration loads -> Version History Drawer automatically updates with new timestamp. ---
Design System ## Color Palette - **Primary:** `#00E5FF` - Active states, markers, CTAs (Cyber cyan) - **Background:** `#050507` - True background (Deepest void) - **Surface (Recessed):** `#0A0B0E` - Sidebars, nav (Sunken dark) - **Surface (Elevated):** `#16181D` - Modals, cards (Raised dark) - **Canvas Backdrop:** `#000000` - Area immediately behind the slide - **Text:** `#E2E4E9` - Primary reading copy - **Muted:** `#828796` - Timestamps, secondary labels, disabled states - **Accent/Warning:** `#FF2A55` - Destructive actions, unresolved critical comments ## Typography - **Headings:** `Space Grotesk`, 600, 20-28px - **UI Labels/Metadata:** `JetBrains Mono`, 400, 12px - **Body:** `Geist`, 400, 14px - **Buttons:** `Geist`, 500, 13px, uppercase tracking **Style notes:** - **Recessed Effect:** Achieved via `box-shadow: inset 0 0 20px rgba(0,0,0,0.8)` on side panels. - **Elevation Effect:** Central canvas gets a sharp `box-shadow: 0 0 0 1px #2A2D35, 0 20px 40px rgba(0,0,0,0.9)`. - **Borders:** Ultra-thin `#1E2128` borders to define rigid structural zones. - **Radii:** Sharp 4px corners, no soft bubbly edges. ## Design Tokens ```css :root { --color-primary: #00E5FF; --color-bg-void: #050507; --color-bg-recessed: #0A0B0E; --color-bg-elevated: #16181D; --color-border: #1E2128; --color-text-main: #E2E4E9; --color-text-muted: #828796; --font-heading: 'Space Grotesk', sans-serif; --font-ui: 'JetBrains Mono', monospace; --font-body: 'Geist', sans-serif; --shadow-recessed: inset 0 0 20px rgba(0,0,0,0.8); --shadow-elevated: 0 0 0 1px #2A2D35, 0 20px 40px rgba(0,0,0,0.9); --radius-sharp: 4px; } ```

Screen Specifications ### Presentation Canvas **Purpose:** Main viewing area for the slide generation. **Layout:** Full viewport. 48px top recessed nav. Central viewing area with an elevated 16:9 slide container. **Key Elements:** - **Top Nav:** `#0A0B0E` background, bottom border `#1E2128`. Contains presentation title, `Share` button, `Export` button. - **Slide Container:** 16:9 aspect ratio, strictly centered. `--shadow-elevated`. No UI elements overlap this unless interacting. - **Action Bar:** Floating bottom-center, `#16181D`, 40px height. Contains `Prev`, `Next`, `Force Refresh`, and `Comments` toggles. **States:** - **Loading:** Slide area displays a wireframe layout pulsing with a `#00E5FF` gradient sweep. - **Error:** Slide area turns solid `#0A0B0E` with a red `#FF2A55` mono-spaced error log. **Interactions:** - **Hover Slide:** Cursor changes to crosshair if Comment mode is active. - **Click `Force Refresh`:** Button icon spins `1s linear infinite`, slide crossfades to loading state. ### Feedback Sidebar **Purpose:** Threaded, coordinate-based comments. **Layout:** 320px fixed-width right panel, full height minus top nav. **Key Elements:** - **Panel Container:** `#0A0B0E` background, `--shadow-recessed`, left border `#1E2128`. - **Comment Card:** `#16181D` background, 4px radius, 12px padding. Displays user avatar, timestamp (`JetBrains Mono`), comment text. - **Input Field:** Fixed at bottom. `#050507` background, 1px `#1E2128` border. Placeholder: `Type / to tag...`. **States:** - **Empty:** "No annotations on this frame." centrally aligned, muted text. **Interactions:** - **Hover Comment Card:** Corresponding marker on the slide canvas pulses heavily with `#00E5FF` glow. ### Version History Drawer **Purpose:** Navigating previous AI generations and manual edits. **Layout:** 280px fixed-width left panel, slides out pushing the canvas. **Key Elements:** - **Timeline List:** Vertical line `#1E2128` connecting node points. - **Version Node:** 12px circular indicator. `#00E5FF` for current, `#1E2128` for past. - **Metadata:** Author/AI label, generation timestamp (e.g., `14:02:44 UTC`). **Interactions:** - **Click Past Version:** Canvas immediately loads selected state without page refresh. Timeline node animates to active state. ### Share Modal **Purpose:** Managing access to the sensitive presentation. **Layout:** 480px centered elevated modal. **Key Elements:** - **Backdrop:** `#050507` at 80% opacity, `backdrop-filter: blur(4px)`. - **Modal Surface:** `#16181D`, `--shadow-elevated`. - **User List:** Rows of users with dropdowns for `Viewer`, `Commenter`, `Editor`. - **Link Toggle:** "Require BlueVoyant SSO" switch. Active state `#00E5FF`. **Interactions:** - **Click Outside:** Closes modal immediately. - **Toggle Switch:** Smooth 150ms translate of the knob, background fades to cyan. ### Export Configurator **Purpose:** Compiling slides to PDF. **Layout:** 400px centered elevated modal. **Key Elements:** - **Scope Selector:** Radio buttons for `All Slides`, `Current Slide`, `Custom Range`. - **Options Toggles:** Checkboxes for `Include Annotations`, `Flatten Layers`. - **Primary CTA:** Full-width bottom button, `#00E5FF` background, `#050507` text, label: `Compile PDF`. **Interactions:** - **Click `Compile PDF`:** Button text changes to `Generating...`, progress bar fills button background horizontally.

Build Guide **Stack:** HTML + Tailwind CSS v3 **Build Order:** 1. **Presentation Canvas:** Establishes the core recessed vs. elevated spatial paradigm. 2. **Top Navigation & Action Bar:** Sets the structural boundaries and interactive framing. 3. **Feedback Sidebar:** Introduces the inner-shadow recessed styling and form inputs. 4. **Share / Export Modals:** Reuses the elevated container logic and implements complex interactive states. 5. **Version History Drawer:** Adds the final layout transition complexity.
